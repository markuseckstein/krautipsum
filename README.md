# KRAUTIPSUM 

Generator for traditional german filler text.

## Running Locally

        npm install
        npm start

Your app should now be running on [localhost:5000](http://localhost:5000/).

## API

Make use of the following API.

        // get random paragraph
        GET /api/kraut

        // get random sentence
        GET /api/sentence

        // get random greeting
        GET /api/greeting

        // get random noun
        GET /api/noun

        // get random verb
        GET /api/verb

        // get random adjective
        GET /api/adjective
